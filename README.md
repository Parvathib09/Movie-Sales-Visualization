🎬 Movie Sales Visualization Dashboard

📌 Project Overview

The Movie Sales Visualization Dashboard is an interactive Power BI project developed to analyze movie industry data and transform raw information into meaningful business insights. This dashboard enables users to explore movie performance through various metrics such as revenue, ratings, genres, audience engagement, and director success.

By leveraging data visualization techniques, the project helps identify trends, patterns, and factors that contribute to a movie's commercial and critical success. The dashboard is designed to support data-driven decision-making and provide a comprehensive understanding of the movie industry's performance.

---

🎯 Objectives

- Analyze movie revenue and financial performance.
- Identify the most successful and popular movie genres.
- Evaluate the relationship between movie ratings and revenue.
- Measure audience engagement through vote counts and popularity metrics.
- Discover top-performing directors and movies.
- Present insights using interactive and visually appealing dashboards.

---

🛠️ Tools & Technologies

- Power BI – Data visualization and dashboard creation
- Microsoft Excel – Data preparation and cleaning
- DAX (Data Analysis Expressions) – Calculated measures and business logic

---

📊 Dashboard Features

- Revenue analysis across movies and genres
- Movie rating and popularity comparison
- Genre-wise performance visualization
- Audience engagement analysis using vote counts
- Director performance evaluation
- Interactive filters and slicers for detailed exploration
